# מפיקסל לקוד: יצירת פלטות CSS מתמונות

מעצבים ומפתחים לעיתים קרובות מתמודדים עם אותה אתגר: למצוא את קבוצת הצבעים המושלמת שעובדת בהarmonia. יש לכם תמונה עם vibe שאתם אוהבים, אבל להפוך אותה לפלטת צבעים פונקצional לאתר לוקח זמן. [מחולל הפלטות CSS](/tools/dev-tools/css-palette-gen) שלנו מאוטomates את התהליך, והופך ויזuals לקוד production-ready.

## למה מפתחים אוהבים את הכלי
במקום לדגום צבעים ידנית ולכתוב ערכי HEX, הכלי שלנו עושה את העבודה הכבדה:
* **מוכן לשימוש:** אנחנו מייצאים את הפלטה בפורמטים כמו CSS variables, JSON, ו-snippets של Tailwind config.
* **דיוק:** האלגוריתם שלנו מזהה את הצבעים הדominant, המשניים וה-accent, ומוודא שהפלטה מאוזנת.
* **יעילות:** חסכו דקות של עבודה מייגעת בכל פעם שמתחילים פרויקט UI חדש.

## איך זה עובד
1. **העלאה:** גררו את התמונה ל[מחולל הפלטות CSS](/tools/dev-tools/css-palette-gen).
2. **ניתוח:** הכלי ממפה מיד את הצבעים הבולטים ביותר בתמונה.
3. **העתקה:** בחרו את פורמט הקוד המועדף (CSS, SCSS או Tailwind) ולחצו להעתקה. כל כך פשוט.

## טיפים מקצועיים למעצבי רשת
* **הגדירו היררכיה:** השתמשו בצבע ה-"Dominant" לזהות המותג העיקרית ובצבע ה-"Accent" לכפתורי call-to-action.
* **בדקו ניגודיות:** גם עם פלטה מעולה, תמיד הריצו את הצבעים דרך בודק נגישות כדי לוודא שהטקסט קריא מול הרקע.

## פרטיות קודם כל
אנחנו מאמינים שנכסים יצירתיים צריכים להישאר שלכם. כל הניתוח קורה מקומית בדפדפן. התמונות שלכם אף פעם לא נשמרות או נשלחות לשרת, מה ששומר על עיצובי הפרויקט חסויים.
